Pokemon_affinity_calculator (black and white)

-=-=- Instructions for setup -=-=-

Create venv:

python3 -m venv env

Switching to venv:

source env/bin/activate

Installing django/dependencies:

pip3 install -r requirements.txt


-=-=- Run Django Webserver -=-=-

To run the development server:

./manage.py runserver
